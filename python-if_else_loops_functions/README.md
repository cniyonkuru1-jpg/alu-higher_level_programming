# Python - if/else, loops, functions

This project covers:
- Conditional statements (if, elif, else)
- Loops (for, while)
- Functions in Python
- Basic problem solving using Python control structures

## Files

- 0-positive_or_negative.py: Prints whether a number is positive, zero, or negative
- 1-last_digit.py: Prints the last digit of a number
- 2-print_alphabet.py: Prints lowercase alphabet
- 3-print_alphabt.py: Prints alphabet except q and e
- 4-print_hexa.py: Prints numbers in decimal and hexadecimal
- 5-print_comb2.py: Prints numbers from 00 to 99
- 6-print_comb3.py: Prints combinations of two digits
- 7-islower.py: Checks for lowercase characters
- 8-uppercase.py: Converts string to uppercase
- 9-print_last_digit.py: Prints last digit of a number
- 10-add.py: Adds two integers
- 11-pow.py: Computes power of a number
- 12-fizzbuzz.py: Prints FizzBuzz sequence
