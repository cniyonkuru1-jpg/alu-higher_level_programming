# Python - Import & Modules

This project covers how to import functions and variables from other files in Python.

## Learning Objectives

- How to import functions from another file
- How to use imported functions
- How to create a module
- How to use the built-in function `dir()`
- How to prevent code in your script from being executed when imported
- How to use command line arguments with your Python programs

## Tasks

### 0. Import a simple function from a simple file
* File: `0-add.py`
* Imports `add` function from `add_0.py` and prints `1 + 2 = 3`

### 1. My first toolbox!
* File: `1-calculation.py`
* Imports functions from `calculator_1.py` and prints results of add, sub, mul, div

### 2. How to make a script dynamic!
* File: `2-args.py`
* Prints the number of arguments and lists them

### 3. Infinite addition
* File: `3-infinite_add.py`
* Prints the sum of all arguments passed to the program

### 4. Who are you?
* File: `4-hidden_discovery.py`
* Prints all names defined in a compiled module that don't start with `__`

### 5. Everything can be imported
* File: `5-variable_load.py`
* Imports variable `a` from `variable_load_5.py` and prints its value
