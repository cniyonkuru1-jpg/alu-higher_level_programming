# python-more_data_structures

Higher-level programming project on sets, dictionaries, and
lambda/map/filter/reduce in Python 3.

## Tasks

| File | Description |
| --- | --- |
| `0-square_matrix_simple.py` | Square every value in a matrix |
| `1-search_replace.py` | Replace an element in a new list |
| `2-uniq_add.py` | Sum the unique integers in a list |
| `3-common_elements.py` | Common elements of two sets |
| `4-only_diff_elements.py` | Elements present in only one set |
| `5-number_keys.py` | Number of keys in a dictionary |
| `6-print_sorted_dictionary.py` | Print a dictionary sorted by key |
| `7-update_dictionary.py` | Add or replace a key/value pair |
| `8-simple_delete.py` | Delete a key from a dictionary |
| `9-multiply_by_2.py` | Double every value in a dictionary |
| `10-best_score.py` | Key with the highest value |
| `11-multiply_list_map.py` | Multiply list values using `map` |
| `12-roman_to_int.py` | Convert a Roman numeral to an integer |

## Requirements

- Ubuntu 20.04 LTS, Python 3.8.5
- pycodestyle 2.7.*
- All files start with `#!/usr/bin/python3` and are executable
